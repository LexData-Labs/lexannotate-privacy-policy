Privacy Policy – LexAnnotate Extension

Last Updated: January 2025

1. What Data We Collect

  • Screenshots of annotation workspaces (only when you save)
  • Annotation data (bounding boxes, labels, coordinates)
  • Email address (for authentication)
  • Login session tokens

2. When Data Is Collected

  • When you explicitly save your work
  • When you click the “Test Capture” button
  • When you log in to the extension

3. How Data Is Used

  • Backup and collaboration via LexAnnotate.ai
  • Email used only for authentication
  • No data is sold to third parties

4. Data Storage

  • Local: Downloads folder (screenshots, JSON files)
  • Cloud: LexAnnotate.ai servers (encrypted in transit)
  • Auth tokens stored in browser storage

5. Third-Party Services

  • LexAnnotate.ai backend server
  • No analytics or tracking services

6. Data Sharing

We do not sell or share user data except when required by law or to provide core services.

7. User Rights

  • Delete local data anytime
  • Request account deletion
  • Disable the extension

8. Security

  • HTTPS encrypted communication
  • Secure authentication tokens
  • No passwords stored locally

9. Contact

Email: support@lexannotate.ai

10. Changes to Policy

Policy updates may occur. Continued use means acceptance.
