PRIVACY POLICY - LexAnnotate Extension
======================================
Last Updated: January 2025

1. WHAT DATA WE COLLECT
-----------------------
- Screenshots of annotation workspaces (only when you save)
- Annotation data (bounding boxes, labels, coordinates)
- Email address (for authentication)
- Login session tokens

2. WHEN DATA IS COLLECTED
-------------------------
Data is only collected when:
- You explicitly save your work in an annotation platform
- You click the "Test Capture" button
- You login to the extension

3. HOW DATA IS USED
-------------------
- Screenshots and annotations are uploaded to LexAnnotate.ai for backup and team collaboration
- Email is used for authentication only
- No data is sold to third parties

4. DATA STORAGE
---------------
- Local: Screenshots and JSON files saved to your Downloads folder
- Cloud: Data uploaded to LexAnnotate.ai servers (encrypted in transit)
- Authentication tokens stored locally in browser storage

5. THIRD-PARTY SERVICES
-----------------------
- LexAnnotate.ai (backend server for data sync)
- No analytics or tracking services

6. DATA SHARING
---------------
We do not sell, trade, or share your data with third parties except:
- When required by law
- To provide the service (LexAnnotate.ai backend)

7. USER RIGHTS
--------------
You can:
- Delete local data by clearing browser storage
- Request account deletion by contacting support
- Disable the extension at any time

8. SECURITY
-----------
- All data transmitted over HTTPS
- Authentication via secure tokens
- No passwords stored locally

9. CONTACT
----------
For privacy concerns, contact: support@lexannotate.ai

10. CHANGES TO POLICY
---------------------
We may update this policy. Continued use of the extension constitutes acceptance of changes.
